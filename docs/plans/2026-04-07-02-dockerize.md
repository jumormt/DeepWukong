# Plan: Dockerize DeepWukong

**Epic:** E1: Modernization
**Design:** N/A

## Summary

Create a minimal GPU-enabled Docker image for DeepWukong. Users can mount their own data, run the full pipeline (preprocessing → training → evaluation), or use the provided preprocessed data. Includes a data preparation guide.

**Decisions locked in:**
- Base image: `pytorch/pytorch:2.6.0-cuda12.4-cudnn9-runtime` (already has torch+CUDA, smaller than devel)
- Data mounted as volume at `/workspace/data`, not baked into image
- joern (Java) NOT included in image — documented as external prerequisite
- `sensiAPI.txt` included in image (hardcoded import in symbolizer.py)
- Config can be mounted or use default
